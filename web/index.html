    Connect a GitHub token to refresh

    This page is static, so the Refresh button asks GitHub Actions to re-gather the data. That needs a fine-grained token with Actions: read&write on this repository. It is stored only in this browser (localStorage) and sent only to api.github.com.

    1. Open Fine-grained tokens → Generate new token.
    2. Repository access → Only select repositories → this repo.
    3. Permissions → Repository → Actions → Read and write.
    4. Generate, copy, and paste below.
